# 🏡 NósDois - INICIAR AQUI

## ⚡ Início Rápido

### **Opção 1: Desenvolvimento Local (Recomendado)**

```bash
cd NosDois-main
npm install
npm run dev
```

Então abra: **http://localhost:5173**

### **Opção 2: Produção Local**

```bash
cd NosDois-main
npm install
npm run build
npm start
```

Então abra: **http://localhost:3000**

### **Opção 3: Script Facilitado**

```bash
cd NosDois-main
./start.sh
```

Então abra: **http://localhost:3000**
